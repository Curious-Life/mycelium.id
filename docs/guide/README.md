encrypts everything with keys only you hold, and serves your context back to **any

shoulder.

## The one-paragraph version

> Mycelium is a self-hosted, single-user **cognitive vault**. It runs as one process
> on your hardware. You feed it your data — chat exports, notes, messages, files — and
> it encrypts everything at rest with keys only you hold, embeds and organizes it
> locally, and serves it back to any AI model through the standard **Model Context
> Protocol (MCP)**. There is no cloud, no multi-tenancy, and no autonomous agent
> acting on its own: it's a tool server that answers when asked. Your data compounds
> across every service you use. You switch models freely. **Nobody else can read it.**
