# Pre-launch checklist

Things that are **scaffolded with placeholders** and MUST be filled in before the
repo goes public / V1 launches. Grep the codebase for `BEFORE LAUNCH` to find them.

## Funding (value-for-value)
- [ ] **GitHub Sponsors handle** — set `github: [...]` in `.github/FUNDING.yml`
      (and uncomment it) with the account/org that has Sponsors enabled. Replace
      `YOUR_GITHUB_SPONSORS_HANDLE` in `README.md` (Support section).
- [ ] **Stripe payment link** — set `custom: [...]` in `.github/FUNDING.yml`
      (optional) and replace `YOUR_STRIPE_PAYMENT_LINK` in `README.md`. Or remove
      the Stripe line if not using it.
